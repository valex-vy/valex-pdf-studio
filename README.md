# 📄 Valex PDF Studio

A **100% free, fast, and privacy-friendly PDF converter** for students. Convert images, Word docs, Excel sheets, PowerPoint files, and text to PDF—and back again!

## ✨ Features

✅ **Convert to PDF** - Images (JPG, PNG, GIF) → Word (DOCX) → Excel (XLSX) → PowerPoint (PPTX) → Text (TXT)
✅ **Convert from PDF** - Extract as Images (ZIP) → Text (TXT) → Word (DOCX)
✅ **Merge PDFs** - Combine multiple PDFs into one
✅ **Compress PDF** - Reduce file size for email submission
✅ **100% Client-Side Processing** - Your files never leave your device
✅ **Dark Mode** - Easy on the eyes
✅ **Mobile Responsive** - Works on all devices
✅ **Zero Ads** - Pure functionality, no popups

## 📈 Monitoring Your Analytics

### Google Analytics Dashboard
1. Go to [Google Analytics](https://analytics.google.com)
2. Sign in with your account (ID: `G-EW0G1D2EVL`)
3. View real-time visitors, page views, user engagement, and conversion events

**Tracked Events:**
- `page_view` - When users visit the site
- `tab_switch` - Which feature tabs users interact with
- `pdf_converted` - PDF conversion (with file count)
- `pdf_to_images` - PDF to images extraction
- `pdf_to_text` - PDF to text extraction
- `pdf_to_word` - PDF to Word conversion
- `pdfs_merged` - PDF merging
- `pdf_compressed` - PDF compression
- `theme_toggle` - Dark/Light mode usage

### Real-Time Visitor Count
- Displayed in the **bottom-left badge** on the site
- Data from CountAPI: [https://countapi.xyz/hit/valex-pdf-studio/visits](https://countapi.xyz/hit/valex-pdf-studio/visits)

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3, Vanilla JavaScript
- **PDF Libraries:**
  - jsPDF - PDF creation
  - PDF.js - PDF reading/extraction
  - PDF-Lib - PDF manipulation
  - Mammoth.js - DOCX extraction
  - SheetJS (XLSX) - Excel handling
  - JSZip - ZIP file creation

- **Analytics:**
  - Google Analytics 4 (GA4)
  - CountAPI (Real-time visitor counting)

- **Hosting:** Netlify
- **Domain:** Custom or netlify.app subdomain

## 📊 Dashboard Setup Guide

### Google Analytics Dashboard Steps:
1. Login to [Google Analytics](https://analytics.google.com)
2. Select property "Valex PDF Studio"
3. Navigate to:
   - **Realtime** → See live visitors
   - **Reports** → Engagement overview
   - **Events** → Track conversions
   - **Audience** → User demographics

### Creating a Custom Dashboard:
1. In GA4, click **Dashboards** → **Create New Dashboard**
2. Add cards for:
   - Real-time active users
   - Event count (conversions)
   - Session duration
   - Bounce rate
   - Top pages
